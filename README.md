# SentinelAI
SentinelAI — Offline C# Cybersecurity Assistant

A local AI-powered cybersecurity toolkit built in C#, capable of log analysis, malware detection, misconfiguration scanning, script reverse engineering, and advanced threat intelligence.
Features include YARA rule scanning, AI malware family classification, MITRE ATT&CK mapping, memory dump summarization, and automated incident investigation — all powered by a self-hosted LLM via gpt4free-ts.
Designed for blue teamers, SOC analysts, and cybersecurity engineers requiring offline, high-trust tools.
